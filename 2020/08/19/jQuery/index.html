一、jQuery概述

1.1 简介

jQuery是一款优秀的javascript的轻量级框架之一，封装了dom操作、事件绑定、Ajax等功能。

特别值得一提的是基于jQuery平台的插件非常丰富，大多数前端业务场景都有其封装好的工具可直接使用。

jQuery下载

  • 官网地址：http://jquery.com

小贴士：

  • jquery-xxx.js
    • 开发版本：有良好的缩进和注释格式，方便开发者阅读源代码
  • jquery-xxx.min.js
    • 生产版本：对代码压缩、注释删除，易于网络传输

1.2 自定义JS框架

框架（Framework）是完成某种功能的半成品，抽取重复繁琐的代码，提供简洁强大的方法实现。

2.2 jQuery和JS区别

jQuery虽然本质上也是js，但如果使用jQuery的属性和方法那么必须保证对象是jQuery对象而不是js对象。

通过js方式获取的是js对象，通过jQuery方式获取的是jQuery对象。两者的关系与区别总结如下：

2.2.1 JS对象和jQuery对象的相互转换

js—>jQuery ：$(js对象) or jQuery(js对象)

jQuery—>js：jq对象[索引] or jq对象.get(索引)

2.2.2 页面加载事件

语法：

  • js：window.onload=function(){...}

  • jQuery：$(function(){})

特点：

  • js：只能定义一次，后面的会覆盖前面的
  • jQuery：可以定义多次…

三、jQuery选择器

css、js、jquery选择器语法一样…

3.1 基本选择器

  • 标签（元素）选择器
    • 语法：$("html标签名") 获得所有匹配标签名称的元素
  • id选择器
    • 语法：$("#id的属性值") 获得与指定id属性值匹配的元素
  • 类选择器
    • 语法：$(".class的属性值")获得与指定的class属性值匹配的元素
  • 并集选择器
    • 语法：$("选择器1,选择器2....")获取多个选择器选中的所有元素

使用演示：

3.2 层级关系选择器

  • 后代选择器

    • 语法：$("A B") 选择A元素内部的所有B元素

  • 父子选择器

    • 语法：$("A > B") 选择A元素内部的所有B子元素

使用演示：

3.3 属性选择器

  • 属性选择器：
    • 语法：$("A[属性名='值']")包含指定属性等于指定值的选择器
  • 复合属性选择器：
    • 语法：$("A[属性名='值'][]...") 包含多个属性条件的选择器

使用演示：

3.4 过滤选择器

  • 首元素选择器
    • 语法： :first 获得选择的元素中的第一个元素
  • 尾元素选择器
    • 语法： :last 获得选择的元素中的最后一个元素
  • 偶数选择器（奇数行）
    • 语法： :even 偶数，从 0 开始计数
  • 奇数选择器（偶数行）
    • 语法： :odd 奇数，从 0 开始计数
  • 指定索引选择器
    • 语法： :eq(index) 指定索引元素

四、jQuery的DOM操作

4.1 jQuery操作内容

  • text()：获取/设置元素的标签体纯文本内容

    • 相当于js：innerText属性

  • html(): 获取/设置元素的标签体超文本内容

    • 相当于js：innerHTML属性

4.2 jQuery操作属性

  • val()：获取/设置 （input标签）value属性值

    • 相当于：input的js对象.value属性

  • attr(): 获取/设置元素的属性

  • removeAttr()：删除属性

jQuery在1.6版本之后，提供另一组API prop 通常处理属性值为布尔类型操作

  • 例如：checked selected等

  • prop()：获取/设置元素的属性

  • removeProp()：删除属性

使用示例：

4.3 jQuery操作样式

  1. 直接修改jQuery对象的样式属性

    • 语法：jQuery对象.css()

      • css(样式名) 获取

      • css(样式名,样式值) 设置

        兼容性好：样式名支持：fontSize | font-size

  2. 添加/删除jQuery对象样式

    • 语法：
      • 添加：jQuery对象.addClass()
      • 删除：jQuery对象.removeClass()

  3. 切换jq对象样式

    • 语法：jQuery对象.toggleClass() 无添加，有删除

4.4 jQuery操作元素

  1. $(标签) 创建一个标签
    • $("<a>xx</a>") 相当于 document.createElement(‘li’)
  2. prepend() 在当前元素内部，首部添加子元素
    • 父对象.prepend(子对象)
  3. append() 在当前元素内部，尾部添加子元素
    • 父对象.append(子对象)

  1. empty() 清空子元素 （断子绝孙）

    • 对象.empty()

  2. remove() 删除自己 （满门抄斩）

    • 对象.remove()

五、jQuery事件绑定

jQuery的事件与js的事件的功能和作用一样，只是在使用语法上稍微有些差异。

  • js对象.事件属性=function(){}

  • jq对象.事件函数(function(){})

常见的事件

  1. 点击事件：
    • click()：单击事件
    • dblclick()：双击事件
  2. 焦点事件
    • blur()：失去焦点
    • focus():元素获得焦点。
  3. 鼠标事件：
    • mousedown()：鼠标按钮被按下。
    • mouseup()：鼠标按键被松开。
    • mousemove()：鼠标被移动。
    • mouseover()：鼠标移到某元素之上。
    • mouseout()：鼠标从某元素移开。
  4. 键盘事件：
    • keydown()：某个键盘按键被按下。
    • keyup()：某个键盘按键被松开。
    • keypress()：某个键盘按键被按下并松开。
  5. 改变事件
    • change()：域的内容被改变。
  6. 表单事件：
    • submit()：提交按钮被点击。
  7. 页面加载事件：
    • $(function(){})
