一、CSS概述

概念：层叠样式表（Cascading Style Sheet）

层叠样式: 对同一个标签添加多个不同的样式,所有样式会叠加在一起展示出效果

作用：页面美化和控制布局

CSS的优点：

  • 实现了样式和内容的分离，提高了显示效果和样式的复用性。
  • 降低耦合性，分工更加明确，CSS专门用于美化，HTML专门用于结构搭建。

二、CSS入门

2.1 HTML引入CSS

HTML有三种引入CSS的方式分别是：

  • 行内样式

    • 在标签中使用style属性指定css代码
    • 如：<div style="color:red;">hello,world</div>

  • 内部样式

    • 在head标签内，定义style标签，在style标签体内写css代码

    • <style>
    div{
        color:red;
    }
</style>

2.3 基本选择器

选择器作用：找出页面一组特定的标签

  • 标签选择器：
    • 语法：标签名{css样式}
    • 特点：指定标签
  • 类选择器：
    • .class名{css样式}
    • 特点：分组设置样式
  • ID选择器：
    • #id名{css样式}
    • 特点：唯一

选择器优先级：id > class > 标签

2.4 扩展选择器

选择所有元素：

  • 语法：*{css样式}

并集选择器：

  • 语法：选择器1,选择器2,选择器3....{css样式}

父子选择器：

  • 语法：父选择器 > 子选择器{css样式}

后代选择器：

  • 语法：父选择器 后代选择器{css样式}

属性选择器：

  • 语法：标签[属性名="属性值"]{css样式}

伪类选择器：选择一些元素具有的状态

  • 语法： 元素:状态{}
  • 如： <a>:状态{}
    • 状态：
      • link：初始化的状态
      • visited：被访问过的状态
      • active：正在访问状态
      • hover：鼠标悬浮状态

2.5 CSS属性介绍

字体、文本

  • font-size：字体大小
    • color：文本颜色
    • text-align：对其方式
    • line-height：行高

背景

  • background

边框

  • border：设置边框属性

尺寸

  • width：宽度
  • height：高度

盒子模型：控制布局

  • margin：外边距

  • padding：内边距

    • 默认情况下内边距会影响整个盒子的大小
    • box-sizing: border-box; 设置盒子的属性，让width和height就是最终盒子的大小

  • float：浮动

    • left
    • right

显示属性

  • display
    • block：指定标签为块级元素
    • inline：指定标签为行内元素
    • none：隐藏元素
