JavaScript

一、概述

1.1 概念

概念：一门客户端脚本语言。

  • 运行在客户端浏览器中的，每一个浏览器都有JavaScript的解析引擎
  • 脚本语言：不需要编译，直接就可以被浏览器解析执行。

1.2 功能

功能：可以来增强用户和HTML页面的交互过程，可以控制HTML元素，让页面有一些动态的效果，增强用户体验。

1.3 特点

  • JS源码不需要编译，浏览器可以直接解释运行

  • JS是弱类型语言，js变量声明不需要指明类型

小结：js是弱类型的脚本语言，可以直接在浏览器解释运行

1.4 JavaScript的组成

组成部分 作用
ECMA Script 构成了JS核心的语法基础
BOM Browser Object Model 浏览器对象模型，用来操作浏览器上的对象
DOM Document Object Model 文档对象模型，用来操作网页中的元素（标签）

二、基础语法

2.1 HTML引入JS

在HTML中引入JS有两种方式：

  • 内部脚本
    • 语法：<script type="text/javascript"> js代码 </script>
    • 作用范围：当前页面
  • 外部脚本
    • 语法：<script type="text/javascript" src="外部js文件路径"></script>
    • 作用范围：所有引入当前JS文件的页面

小贴士：

  • 扩展:在开发时 style和link标签通常在head头部，script标签在body尾部…

  • 注意：如果script标签引入了外部脚本，那么标签体不能自闭合，而且不允许有js代码

2.2 注释

  • 单行注释：//注释内容
  • 多行注释：/* 注释内容 */

2.3 JS的三种输出方式

在JavaScript中，有三种输出方式，分别是

  • 在浏览器弹框输出：alert('内容');

  • 输出到html页面：document.write('内容');

  • 输出到浏览器控制台：cosole.log('内容');

2.4 JS的数据类型及其变量声明

2.4.1 数据类型

数据类型可分为原始数据类型和引用数据类型：

  • 原始数据类型
    • number：数值。包括整数、浮点数、NaN(not a number 一个不是数字的数字类型)
    • string：字符串。
    • boolean：true、flase
    • undefine：未定义
    • null：一个对象为空的占位符
  • 引用数据类型：对象(Object)

小贴士：

  • typeof 变量名：可以查看变量的数据类型

2.4.2 变量声明

Java语言是强类型语言，而JavaScript是弱类型语言。

  • 强类型：在开辟变量存储空间时，定义了空间将来存储的数据的数据类型。只能存储固定类型的数据
  • 弱类型：在开辟变量存储空间时，不定义空间将来的存储数据类型，可以存放任意类型的数据。

定义语法(这里讲解ES6的语法)：

  • 变量：let 变量名 = 变量值;

  • 常量：const 常量名 = 常量值;

2.5 JS的运算符

  1. 一元运算符：只有一个运算数的运算符
    • ++，–，+【正号】
  2. 算数运算符
    • +，-，*，/，%…
  3. 赋值运算符
    • =，+=，-=
  4. 比较运算符
    • <，>，>=，<=，==，===(全等于)
  5. 逻辑运算符
    • &&，||，!
    • 其他类型转boolean：
      1. number：0或NaN为假，其他为真
      2. string：除了空字符串(“”)，其他都是true【双引号内有空格不算空串】
      3. null&undefined:都是false
      4. 对象：所有对象都为true
  6. 三元运算符
    • ? :
    • 语法：
      • 表达式? 值1:值2;
      • 判断表达式的值，如果是true则取值1，如果是false则取值2；

js与java的运算符使用方法大致相同。

NaN：not a number（意思是不是一个数字）

需要注意的是:

  • js的数值可以与字符串参与数学运算，底层进行了隐式转换

  • ===：全等于，数值和数据类型全部一样才为true，==只比较数值

三、JS函数

js函数是执行特定功能的代码块.也可以称为js方法

3.1 普通函数

格式：

调用：函数名(实际参数1，实际参数2...);

js函数特点：

  • 声明的函数不需要定义返回值
  • 声明的函数形参不需要使用变量修饰符声明
  • 如果有返回值就加上return
  • js没有方法重载，重名的会被覆盖…
  • js的函数参数个数不太严谨【了解】
  • 所有的函数体内置了arguments数组，用于接收调用时传递的实际参数
  • js函数支持可变参数

四、事件

事件：JS可以监听用户的行为,并调用函数来完成用户交互功能.

4.1 常用事件

点击事件：

  • onclick：单击事件
  • ondbclick：双击事件

焦点事件：

  • onblur：失去焦点
  • onfocus：元素获得焦点

加载事件：

  • onload：页面加载完成后立即发生

鼠标事件：

  • onmousedown 鼠标按钮被按下。
  • onmouseup 鼠标按键被松开。
  • onmousemove 鼠标被移动。
  • onmouseover 鼠标移到某元素之上。
  • onmouseout 鼠标从某元素移开。

键盘事件：

  • onkeydown 某个键盘按键被按下。

  • onkeyup 某个键盘按键被松开。

  • onkeypress 某个键盘按键被按下并松开。

改变事件：

  • onchange 域的内容被改变。

表单事件：

  • onsubmit 提交按钮被点击。

五、JS内置对象

5.1 string对象

字符串对象

  • js构造字符串对象可以使用单引号('')，双引号("")，反引号(``)三种方式

字符串的常用方法：

  • substring(startIndex,endIndex) [start,end) 字符串截取
  • trim() 去掉左右空格
  • split() 切割字符串，变为数组

5.2 array对象

数组对象：

构造数组对象有两种方式

  • let arr = [ele1,ele2,ele3];
  • let arr = new Array(ele1,ele2,ele3);

特点：js数组长度和类型可以任意，所以我们可以简单理解为数组就是java中的list集合

数组的常用方法：

  • 添加元素

    • push() 向尾部添加
    • unshift() 向头部添加

  • 删除元素

    • pop() 弹出最后一个元素
    • shift() 弹出第一个元素

  • 合并数组

    • concat()

  • 数组排序

    • sort()

  • 将数组转为字符串 （与字符串对象.split()方法相反 ）

    • join()

  • 数组反转

    • reverse()

5.3 Date对象

日期对象：let date = new Date();

常用方法：

  • 获取年份：getFullYear()

  • 获取月份：getMonth()

  • 获取几号：getDate()

  • 获取1970-至今的毫秒值：getTime()

  • 等等….

5.4 Math对象

数学运算对象：

  • 四舍五入：round()
  • 向下取整：floor()
  • 向上取整：ceil()
  • 产生随机数：random()【返回[0,1])之间的随机数，左闭右开，包含0不包含1】

可以理解为Math对象为Java中的数学工具类Math,里面的方法直接通过Math.方法名调用即可.

5.5 全局函数

全局函数：不需要创建对象，可以直接调用的函数，称为全局函数。

常用：

  • 字符串转为数字、

    • parseInt() 转为整型

    • parseFloat()转为浮点型

      注意：从第一个字符开始转换，遇到非数值字符停止转换

  • 对字符串编码和解码

    • encodeURI()编码
    • decodeURI() 解码

  • 把字符串当作js表达式来执行

    • eval("js代码")

六、BOM对象

浏览器对象模型：Browser Object Model

作用：把浏览器抽象成为一个对象模型,我们可以使用js模拟浏览器的一些功能。

6.1 window对象

window对象：是当前窗口的一个顶级对象

6.1.1 三种弹框

警告(提示)框：alert()

确认框：confirm()

  • 点击确定 返回true，点击取消 返回false

输入框：prompt()

  • 点击确定返回 用户输入值，点击取消返回 null

6.1.2 二种定时器

周期性定时器：

  • 创建：let interval = setInterval('js代码',毫秒值)
  • 销毁：clearInterval(interval);

一次性定时器

  • 创建：let timeout = setTimeout('js代码',毫秒值);
  • 销毁：clearTimeout(timeout);

6.2 Location对象

把浏览器的地址栏封装为location对象

Location地址

  • 获取当前窗口地址
    • ocation.href
  • 刷新当前窗口
    • location.reload()
  • 当前页跳转
    • location.href='新地址'

扩展：如果想打开一个新标签页，跳转新的页面

  • window.open('新地址')

七、DOM对象

文档对象模型：Document Object Model

作用：把页面中所有标签抽象为一个Document对象，然后通过JS动态修改标签及属性内容。

7.1 DOM获取元素

第一种方式(ES6之前)：

  • 获取一个：document.getElementById(id属性值)
  • 获取多个：
    • document.getElementsByTagName(标签名)：返回数组
    • document.getElementsByClassName(class属性值)：返回数组
    • document.getElementsByName(name属性值)：返回数组

第二种方式：

  • 获取一个：document.querySelector(id选择器)
  • 获取多个：document.querySelectorAll(css选择器)：返回数组

获取一个推荐使用getElementById，获取多个推荐使用ES6的语法。

7.2 DOM操作内容

  1. 获取或者修改元素的 纯文本内容

    • 语法：js（元素）对象.innerText;

  2. 获取或者修改元素的 html内容（可以解析标签）

    • 语法：js（元素）对象.innerHTML;

  3. 获取或者修改包含自身的html内容（了解）

    • 语法：js（元素）对象.outerHTML;

7.3 DOM操作属性

  1. 获取文本框的值,单选框或复选框的选中状态

    • 语法:
      • js对象.属性名 获取指定属性值
      • js对象.属性名='xxx' 设置指定属性的新值

  2. 给元素设置自定义属性使用:

    • 语法: js对象.setAttribute(属性名,属性值)

  3. 获取元素的自定义属性值
    语法: js对象.getAttribute(属性名)

  4. 移除元素的自定义属性（了解）
    语法:js对象.removeAttribute(属性名)

7.4 DOM操作样式

  1. 给标签的style属性设置一个css样式

    • 语法: js对象.style.样式名='样式值'
    • js的样式名：fontSize

  2. 给标签的style属性批量设置css样式

    • 语法:js对象.style.cssText='css样式'
      • css的样式名：font-size

  3. 给标签绑定class的类选择器

    • 语法: js对象.className='class选择器名'

7.5 DOM操作元素

  1. 创建一个标签对象

    • 语法：document.createElement('标签名称')

  2. 给父标签添加一个子标签

    • 语法：父标签对象.appendChild(子标签对象)

八、JS正则

作用：根据定义好的规则，过滤文本内容；这里我们用于校验表单

JS中的正则：

  • 创建：
    • let rege = new RegExp(“正则表达式字符串”);
    • let rege = /正则表达式/;
  • 验证方法
    • 正则对象.test(要验证的字符串)
      • 符合正则规则就返回true，否则false
